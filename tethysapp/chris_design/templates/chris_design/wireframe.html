Wireframe

Utah Snowmelt App

We developed a preliminary design for the app layout, which will most likely be simplified due to the difficulty of finding desired data. I have included the original text from the mockup assignment.


Main Page



The principal window on the main page will be a map of the Utah area. Users will be able to manually pan and zoom around the area. On the side menu, users will be able to toggle the base map, and different layers on and off. Users will also be able to select differing snow data and watershed boundaries within these layers. A zoom tool will be added with a drop down menu that will allow users to select a specific region, or gauge within said region. A latitude and longitude search bar will also be added. Finally, the users will be able to view historical data by selecting any date. In the top right corner, a link to the information window and a help window will be included.


Pop-Up



When users select a specific basin or gauge, this window will appear. The graph will display either the snow-water equivalent, snow depth, or snow cover of the current year and the historical median. A table will also be generated with each of these three options broken down by month (from December to June).


Information Page



This window will appear when users click on the info icon on the main page. It will describe the purpose of the app, give some tips on how to use it, and provide citations.
